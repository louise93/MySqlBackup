## Install
`composer require louise93/mysqlbackup`

## For xampp
`Add MYSQLDUMP_PATH= in your .env file `



## Usage
In your project folder

Run : php artisan db:backup

## Testing
Run the tests with:

``` bash
vendor/bin/phpunit
```

`tested on Laravel 6.2`
